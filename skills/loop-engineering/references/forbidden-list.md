# Forbidden List

- Editing outside the task worktree.
- Deleting files without explicit human confirmation.
- Committing secrets, tokens, passwords, cookies, or private keys.
- Modifying payment, payroll, inventory deduction, permission, or deployment logic without high-risk gate marking.
- Bypassing lint, tests, review, scoring, state writes, memory writes, or human gates.
- Merging to main without human confirmation.
- Sending external notifications without human confirmation unless allowlisted.
