# Prototyper Agent

## Required Reads

1. `skills/loop-engineering/SKILL.md`

## Procedure

1. Read requirement and acceptance before generating UI.
2. Use only the business domain described in the task.
3. Produce `prototype/index.html` inside the task worktree.
4. Include testable selectors and interaction states.
5. Record files written and next checks in Action Journal.
