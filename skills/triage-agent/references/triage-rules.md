# Triage Rules

## Task Classes

- PRD_GAP
- DEV
- BUG
- TEST
- SECURITY_PERMISSION
- RELEASE_GATE
- OPS_DOCS

## Priority Levels

- P0: Production or data safety risk.
- P1: Blocks core delivery.
- P2: Important but not blocking current release.
- P3: Cleanup or improvement.

## Output

- Task ID.
- Class.
- Priority.
- Assigned next role.
- Required skill files.
- Required tools.
- Risk flags.
